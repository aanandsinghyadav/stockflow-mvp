# StockFlow MVP

A minimal multi-tenant SaaS inventory management system built as part of a technical assessment.

## Tech Stack

### Backend
- Java Spring Boot
- Spring Security (JWT)
- Spring Data JPA

### Frontend
- Next.js (React)
- Tailwind CSS

### Database
- MySQL (Database)

## Features (MVP)
- User authentication with organization context
- Product CRUD
- Inventory tracking
- Low stock dashboard

## Status
- Initial setup in progress
